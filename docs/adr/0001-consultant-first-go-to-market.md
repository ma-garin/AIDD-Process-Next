# ADR-0001: 最初のターゲットをコンサルタント/SIerに絞る

- **Status**: Accepted
- **Date**: 2026-06-16

## Context

AIDD Process Nextを有償サービスとして展開するにあたり、最初の顧客セグメントを決定する必要があった。候補はエンドユーザー直販（QAリード・CTO）とコンサルタント経由の2択。

## Decision

最初のターゲットを**コンサルタント/SIer**とし、代理店モデルで展開する。

## Consequences

**メリット:**
- 1コンサルタント契約で複数クライアントへの普及が実現する
- コンサルタントの「営業資材」としてツールが機能し、普及速度が上がる
- AIIDDブランド認知ゼロの状態でのエンドユーザー直販より成約確率が高い

**デメリット・トレードオフ:**
- エンドユーザーの声が間接的になり、プロダクト改善フィードバックにノイズが入る
- コンサルタントが「競合ツール」として扱わない信頼関係の構築が必要
- エンドユーザー向けセルフサービス機能の優先度が下がる
