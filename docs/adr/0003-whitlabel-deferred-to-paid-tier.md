# ADR-0003: ホワイトラベルをフェーズCでは提供せず有料機能化する

- **Status**: Accepted
- **Date**: 2026-06-16

## Context

コンサルタントが顧客に診断を提供する際、「AIDD Process Next」ブランドが前面に出ることへの抵抗感が生まれる可能性がある。URLクエリパラメータでブランドカラー・ロゴを注入するホワイトラベル機能の実装は技術的には低コストで可能。

## Decision

フェーズCではホワイトラベルを提供しない。コンサルタントはAIDDブランドのまま顧客に提供する。ホワイトラベルはフェーズA以降の有料差別化機能とする。

## Consequences

**メリット:**
- フェーズCの実装スコープを絞り、MVPを早く届けられる
- コンサルタントがAIDDブランドを顧客に見せることで、AIDDの認知拡大が進む
- ホワイトラベルを有料機能にすることで、フェーズA課金移行の動機になる

**デメリット・トレードオフ:**
- ブランドへのこだわりが強いコンサルタントはフェーズCで採用しない可能性がある
- 競合ツールがホワイトラベルを無料提供している場合、差別化が逆転する
