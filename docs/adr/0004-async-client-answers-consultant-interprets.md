# ADR-0004: クライアントが事前に非同期で回答し、コンサルタントセッションは解釈に集中する

- **Status**: Accepted
- **Date**: 2026-06-16

## Context

コンサルタントがクライアントと診断を進めるセッション設計を検討した。候補は:
- A: コンサルタントが同席しながら一緒にリアルタイムで回答する
- B: コンサルタントがURLを送り、クライアントが自分で回答して結果を共有する
- C: コンサルタントが代わりに入力し、レポートを納品する

50問を対面でこなすと平均2時間以上かかり、コンサルタントの時間単価を考えると非効率。

## Decision

クライアントが事前に非同期で50問を回答し（URLを送って宿題形式）、コンサルタントとのセッションはスコアの解釈・優先度付け・改善計画の議論に集中する。

## Consequences

**メリット:**
- コンサルタントの時間を「価値の高い解釈・提言」に集中できる
- セッション時間を短縮でき、クライアントのスケジュール調整が容易になる
- クライアントが一人で回答することで、正直な自己評価が得やすい（コンサルタントの目を気にしない）

**デメリット・トレードオフ:**
- クライアントが事前回答を怠るとセッションが成立しない（宿題の徹底が必要）
- 回答中に生まれる「この質問の意味は？」という疑問をリアルタイムで解消できない
- URLエクスポート機能（ADR-0002）が前提条件になる
